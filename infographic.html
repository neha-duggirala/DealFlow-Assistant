Multi-Agent Human-in-the-Loop Architecture

The DealFlow Assistant utilizes a LangGraph-based agentic workflow. Unstructured communications are embedded and stored in a Vector Database to maintain deal context. The LLM analyzes the sentiment and status, but critical actions (like sending negotiation emails) require a Human-in-the-Loop (HITL) approval.

LangGraph Implementation Code

Python implementation showing the StateGraph definition. Notice the interrupt_before=["human_review"] configuration, which enables the Human-in-the-Loop architecture by pausing the graph execution until the SBER specialist approves the AI's generated task. 

from typing import TypedDict, Annotated, List
from langgraph.graph import StateGraph, END
from langchain_community.chat_models import GigaChat
from langchain_core.messages import HumanMessage, AIMessage

class DealState(TypedDict):
    deal_id: str
    communications: List[str]
    context_retrieved: str
    analysis_result: str
    proposed_task: str
    human_approved: bool

def retrieve_context(state: DealState):
    db_context = "Retrieved past negotiations regarding logistics."
    return {"context_retrieved": db_context}

def analyze_communications(state: DealState):
    llm = GigaChat(credentials="YOUR_GIGACHAT_AUTH_DATA", verify_ssl_certs=False)
    prompt = f"Analyze communications: {state['communications']}. Context: {state['context_retrieved']}."
    response = llm.invoke([HumanMessage(content=prompt)])
    return {"analysis_result": response.content, "proposed_task": "Draft email to client regarding logistics delay."}

def human_review(state: DealState):
    print(f"REVIEW NEEDED FOR DEAL {state['deal_id']}: {state['proposed_task']}")
    return state 

def execute_task(state: DealState):
    if state.get("human_approved", False):
        print("Task executed and synced to CRM.")
    return {"deal_id": state["deal_id"]}

workflow = StateGraph(DealState)

workflow.add_node("retrieve_context", retrieve_context)
workflow.add_node("analyze", analyze_communications)
workflow.add_node("human_review", human_review)
workflow.add_node("execute", execute_task)

workflow.set_entry_point("retrieve_context")
workflow.add_edge("retrieve_context", "analyze")
workflow.add_edge("analyze", "human_review")
workflow.add_edge("human_review", "execute")
workflow.add_edge("execute", END)

app = workflow.compile(interrupt_before=["human_review"])
